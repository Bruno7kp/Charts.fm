# Charts.fm

## Project setup
```
npm install
```

### Set yout last.fm api key on .env
```
VUE_APP_LAST_FM_KEY="Your api key here"
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Run your unit tests
```
npm run test:unit
```

### Run your end-to-end tests
```
npm run test:e2e
```
